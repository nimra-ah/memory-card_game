# Memory Card Game

A memory card game built with React using the PokéAPI.

## Features

* Fetches Pokémon data from the PokéAPI
* Displays 12 Pokémon cards
* Shuffles cards after each click
* Tracks current score
* Tracks best score
* Resets the game when the same card is clicked twice

## Built With

* React
* Vite
* JavaScript
* CSS
* PokéAPI

## Getting Started

```bash
npm install
npm run dev
```
